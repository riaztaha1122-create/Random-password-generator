# Simple Authentication System (Login/Signup)

This is a simple authentication system built with **HTML, CSS, and JavaScript**. It features a beautiful, modern UI and allows you to sign up and log in. All user data is stored in your browser (no internet or backend required).

## How to Use

1. **Open the Project**
   - Go to the `auth-system` folder.
   - Double-click `index.html` to open it in your browser.

2. **Sign Up**
   - Click the **Sign Up** button.
   - Enter your name, email, and password (at least 6 characters).
   - Click **Sign Up**. If successful, you’ll see a message.

3. **Sign In**
   - Click the **Sign In** button.
   - Enter your registered email and password.
   - Click **Sign In**. If successful, you’ll see a welcome message.

## Notes
- All data is saved in your browser’s local storage. If you clear your browser data, accounts will be lost.
- This is for learning/demo purposes only. Do not use for real accounts or sensitive data.

## For Beginners
- No coding knowledge is needed. Just open `index.html` and follow the instructions above.
- You can change the look by editing `style.css`.

Enjoy your beautiful authentication system!
